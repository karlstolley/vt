A Plan for Action

A Plan for Action

  1. Take your writing open source: Learn XHTML, CSS, and JavaScript, and compose as many things as possible with them.
  2. Upgrade your operating system: Install and use Linux or a Unix-like OS (e.g., Mac OS X); become familiar with the command line.
  3. Make history: Install and configure Git, and put all of your open-source writing projects under Git control.
